AnchorLine

new AnchorLine(attrs)

带有拖拽锚点的 Line 允许闭合为多边形
Parameters:
Name Type Description
attrs Object 详情参数参考Konva.Circle
Properties
Name Type Description
id String id
name String name
points Array 坐标点数据
stroke String 画笔颜色 仅初始化时可设置
strokeWidth Number 画笔宽度 仅初始化时可设置
hitStrokeWidth Number 点击识别范围 仅初始化时可设置
fill String 填充颜色
dash Array 虚线数组 详情参照konva.Line 仅初始化时可设置
closed Boolean 是否闭合图形 仅初始化时可设置
draggable Boolean 是否可拖拽
anchor Boolean 是否绘制拖拽锚点 默认为true 仅初始化时可设置
anchorVisible Boolean 拖拽锚点是否显示默认 true, anchor 为true时有效
absoluteSize Boolean 绝对尺寸，与舞台一同缩放 默认true 仅初始化时可设置
adsorb Boolean 是否开启锚点吸附
Example
let node = new KMaps.AnchorLine({...})
